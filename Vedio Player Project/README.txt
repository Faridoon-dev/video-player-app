# Video Player App

A custom video player built using HTML, CSS, and JavaScript.

## Features

- Play and pause video
- Forward video
- Rewind video
- Restart video from the beginning
- Clean and simple user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Skills Demonstrated

- Video API
- DOM Manipulation
- Event Handling
- Media Controls
- Interactive User Interface

## Purpose

This project was created to practice JavaScript by building a custom video player and working with HTML5 video controls.
